# KeyPilot

KeyPilot is a mobile‑first responsive web app for managing the home‑buying journey in New Zealand.

The MVP focuses on helping buyers track:
- property purchase workflows
- conditional deadlines
- tasks
- documents
- key contacts

## Architecture

Frontend
- Next.js
- TypeScript
- TailwindCSS

Backend
- .NET 10 Modular Monolith API

Data
- Postgres

Storage
- S3 compatible object storage

Notifications
- Email reminders

## Project Goal

Help buyers manage the process from **offer acceptance → settlement** without the chaos of email, spreadsheets, and PDFs.

## Local Ports

- Postgres host port: `5433`
- API base URL: `http://localhost:8081`
- Frontend URL: `http://localhost:3000`
