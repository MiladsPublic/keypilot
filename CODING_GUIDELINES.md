# Coding Guidelines

## Frontend

Framework: Next.js
Language: TypeScript

Rules:
- Mobile first UI
- TailwindCSS styling
- shadcn/ui components
- Clean component structure

## Backend

Runtime: .NET 10

Architecture:
- Vertical slice architecture
- Feature-based modules

Patterns:
- Commands and queries
- REST APIs
- DTO responses

## Database

- Postgres
- snake_case tables
- migrations via EF Core
